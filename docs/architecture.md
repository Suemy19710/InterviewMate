# InterviewMate - System Architecture 

## 1. Overview 
InterviewMate is an AI-powered, context-aware interview preparation platform.  
It helps candidates practice interviews tailored to **their resume (CV)** and **specific job descriptions (JD)** from companies.

Core capabilities: 
- resume upload and parsing (pdf/docx)
- job description parsing and requirement extraction 
- cv-jd semantic matching 
- personalized interview question generation
- context-aware answer scoring with rubric-based feedback 
- user dashboard with history and progress
